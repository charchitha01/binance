# Binance Futures Testnet Web App

A Flask-based web application that allows users to place BUY and SELL Market and Limit orders on Binance Futures Testnet using a simple web interface.

## Features

- Place Market and Limit orders
- Supports BUY and SELL positions
- Connects to Binance Futures Testnet
- Secure API key management using .env file
- Input validation and error handling
- Displays full order response on website
- Logging of requests and responses
- Simple and beginner-friendly web interface

---

## Tech Stack

- Python 3.x
- Flask
- python-binance
- HTML
- Binance Futures Testnet API
